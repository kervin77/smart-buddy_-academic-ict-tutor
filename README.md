IGCSE ICT Interactive Learning Platform

Property of Vardin Infinity Ltd

Project Overview

This is a professional educational application designed to support Year 10 students following the IGCSE/GCSE ICT curriculum. The platform utilizes Google's latest Gemini 2.5 models to provide a multimodal learning experience, featuring both expert academic text generation and real-time neural speech synthesis.

Key Technologies

Text Engine: gemini-2.5-flash-preview-09-2025 (Optimized for low latency and high accuracy).

Speech Engine: gemini-2.5-flash-preview-tts (Providing natural, prosody-rich voice interaction).

Academic Level

Target Audience: Year 10/11 Students (Ages 14-16)

Curriculum Alignment: Pearson Edexcel / Cambridge IGCSE ICT

Tone: Professional, Academic, Encouraging
